# Contributing to libwdi-ext

Thank you for considering contributing!

## How to contribute

### 1. Fork & Branch
- Fork `libwdi-ext`
- Create feature branches such as:
  - `feature/ext-inf`
  - `feature/zadig-ui`
  - `bugfix/win11-install`

### 2. Coding Style
- Match the upstream libwdi C style
- English-only comments
- No trailing whitespace
- Prefer minimal diffs

### 3. PR requirements
- Describe the change & motivation
- Include testing notes (Win7 + Win10/11)
- Update changelog when appropriate
- No proprietary files allowed (INF/SYS/CAT)

### 4. Legal
- libwdi-ext is GPLv3
- Do not include:
  - FTDI, ST, Prolific, etc. drivers
  - Microsoft binaries
  - Any proprietary content

### 5. Where to discuss
Issues & discussions welcome via GitHub.

Thank you!
